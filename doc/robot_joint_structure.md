# 机器人关节结构文档

## 概述

这是一个具有**19自由度**的三臂机器人系统：
- 1个自由度：底座平台旋转
- 6个自由度：机械臂A
- 6个自由度：机械臂B
- 6个自由度：机械臂S（特殊臂）

## 整体结构树

```
base_link (固定底座, 289.3kg)
    │
    └─[joint_platform_D1]─> platform_link (旋转平台, 5.6kg)
                                │
                                ├─[joint_platform_A1]─> 机械臂A (6-DOF)
                                │
                                ├─[joint_platform_B1]─> 机械臂B (6-DOF)
                                │
                                └─[joint_platform_S1]─> 机械臂S (6-DOF)
```

---

## 1. 底座系统

### base_link (根链接)
- **类型**: 固定底座
- **质量**: 289.3 kg
- **功能**: 机器人的主要固定平台

### joint_platform_D1
- **父链接**: base_link
- **子链接**: platform_link
- **类型**: 旋转关节 (Revolute)
- **旋转轴**: Z轴 [0, 0, -1]
- **运动范围**: -3.14 ~ 3.14 弧度 (360°)
- **功能**: 允许整个平台及三个机械臂一起旋转

### platform_link (旋转平台)
- **质量**: 5.6 kg
- **功能**: 承载三个机械臂的旋转平台

## 6. 命名规则

### 关节命名
- **格式**: `joint_platform_[ARM][NUMBER]`
- **ARM**: A, B, S (机械臂标识) 或 D (平台旋转)
- **NUMBER**: 1-6 (关节编号)

### 链接命名
- **格式**: `Link_[ARM][NUMBER]`
- **ARM**: A, B, S
- **NUMBER**: 1-6

## 7. 设计用途分析

### 机械臂A和B
- 对称布置在平台前方
- 标准6自由度操作臂
- 适合协同操作和双臂任务
- 大范围运动能力

### 机械臂S
- 位于平台中央下方
- 具有单向关节限制（S2、S3）
- 腕部运动范围受限
- 可能用于特殊任务（如支撑、精密操作等）

### 平台旋转
- 提供额外的全局自由度
- 扩大三个机械臂的工作空间
- 允许360°全方位操作
